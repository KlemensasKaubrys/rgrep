# rgrep
## Gnu grep alternative written in rust
### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/KlemensasKaubrys/rgrep --depth 1
   ```

2. Navigate to the project directory:
   ```bash
   cd rgrep
   ```

3. Install dependencies, and build project:
   ```bash
   cargo build
   ```
